# Snake Game 🐍

This is a simple snake game created using Python.

## Features
- Score system
- Game Over logic

## How to Run
Run the main file to start the game.

## 📌 Project Description

This is a classic Snake Game developed using Python and Pygame.  
The project uses core programming concepts like game loops, collision detection, event handling, and score tracking.

Key Implementations:
- Real-time keyboard controls
- Dynamic snake movement
- Collision detection with walls and self
- Score tracking system
- Game over logic

This project highlights my understanding of logic building and structured programming.
